# Lottie hero animations

Designed vector animations rendered by the content-engine studio (via
`LottieObject`, which falls back to a CSS object if a file is absent).

Current slots (served at `/lottie/<name>.json`):

| File              | Used by                  | Scene                                  |
| ----------------- | ------------------------ | -------------------------------------- |
| `reel-hero.json`  | Reel · smart screening   | investor thinking where to invest      |
| `filter.json`     | Reel · smart screening   | quality-filter funnel mechanism        |
| `crisis-dip.json` | Reel · smart screening   | market-dip crash scene                 |
| `story-hero.json` | Story · hype check       | pump-and-dump shock scene              |

Provenance: free-licensed LottieFiles library animations, supplied and placed
here by the repo owner (who downloaded them and holds the license). All four are
pure vector — verified free of embedded images, logos, and trademarks. When
adding more, keep the same bar: generic, abstract, vector, cleanly licensed —
nothing with brand logos, real photos, or trademarked products.
